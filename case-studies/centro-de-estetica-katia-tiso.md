# Centro de Estetica Katia Tiso

## Contexto

Este projeto foi estruturado como uma landing page mobile-first para um negocio local com necessidade de captar leads de forma direta, com pouca friccao e baixo custo de publicacao.

## Desafio

O problema central era transformar uma pagina institucional em uma ferramenta de captacao. Isso exigia:

- jornada curta;
- CTA recorrente;
- adaptacao forte para mobile;
- facilidade para evoluir copy e oferta;
- possibilidade de integrar formulario e WhatsApp sem reescrever a base.

## Abordagem adotada

A solucao foi manter o projeto estatico e sem dependencia de build, priorizando:

- publicacao simples;
- carregamento rapido;
- manutencao objetiva;
- separacao entre conteudo, configuracao e renderizacao.

Essa escolha faz sentido quando o objetivo e colocar a oferta no ar com rapidez e manter espaco para evolucao posterior.

## Decisoes tecnicas relevantes

- Uso de HTML, CSS e JavaScript modular para evitar complexidade desnecessaria.
- Conteudo centralizado em arquivo proprio para ajustes de copy.
- Arquivo de configuracao separado para integracoes e canal de contato.
- Captura de UTM e fallback para WhatsApp como parte do fluxo de lead.

## Impacto observavel

Nao ha metrica publica vinculada a este case. O valor demonstrado aqui esta na forma como a base foi estruturada para conversao, publicacao simples e evolucao futura sem acoplamento excessivo.

## O que este case prova sobre a Nexus7

- Capacidade de transformar necessidade comercial em arquitetura enxuta.
- Criterio para escolher stack proporcional ao problema.
- Entendimento de captacao digital alem da camada visual.
