# Estudos de caso

Os estudos de caso aprofundam os projetos publicados. Aqui a Nexus7 mostra raciocinio, criterio tecnico e contexto de negocio.

Cada estudo deve deixar claro:

- desafio;
- contexto operacional;
- abordagem;
- decisoes de arquitetura;
- impacto observavel ou limitacoes publicas.

## Regra principal

Se nao houver metrica publica, autorizacao do cliente ou prova verificavel, documente o que pode ser provado sem inventar numero nem depoimento.
