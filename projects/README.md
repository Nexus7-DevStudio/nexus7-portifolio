# Projetos publicados

Esta pasta guarda resumos executivos dos projetos que ajudam a demonstrar a capacidade tecnica da Nexus7.

Cada projeto deve responder de forma rapida:

- o que foi construido;
- qual problema foi resolvido;
- quais decisoes tecnicas importaram;
- por que esse trabalho e relevante para um potencial cliente.
