# Centro de Estetica Katia Tiso

- Tipo: Landing page de captacao
- Status: Publico
- Repositorio publico: [CristianoRFB/landing-page-clinica-estetica](https://github.com/CristianoRFB/landing-page-clinica-estetica)

## Contexto

Projeto criado para captar agendamentos com baixa friccao, priorizando acesso mobile e conversao via WhatsApp e formulario.

## Solucao

Implementacao estatica sem dependencias de build, com HTML, CSS e JavaScript modular. O conteudo fica centralizado em arquivo de dados, enquanto a configuracao de integracoes permanece separada para permitir ajustes sem retrabalho estrutural.

## Decisoes que importam para o cliente

- Publicacao simples e barata.
- Baixo atrito para ajustes de copy e oferta.
- Base facil de evoluir para API, CRM ou automacao.
- Estrutura orientada a CTA recorrente, prova social e captacao de lead.

## Stack

- HTML
- CSS
- JavaScript modular

## O que este projeto demonstra

- Capacidade de construir paginas orientadas a conversao.
- Organizacao de front-end estatico com manutencao simples.
- Pensamento de negocio aplicado a aquisicao de leads.
